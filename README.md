# Bookstore Frontend

Frontend Application for managing a bookstore.
Developed as part of the Web Development course.

## Features

- **Authentication**: Login with Username/Password
- **Book Management**: View, Add, Edit, Delete books
- **Dashboard**: Visualize book statistics and stock levels
- **AI Integration**: Ask AI (Google Gemini) for book summaries in Thai

## Tech Stack

- React
- Vite
- Ant Design
- ApexCharts / Recharts
- Axios

## How to Run

1. `npm install`
2. `npm run dev`
